# ShopEasy - E-commerce Website

A fully responsive online store with product listings, shopping cart functionality, and checkout process.

## Features

- Product browsing with category filtering
- Shopping cart with add/remove/update quantity functionality
- Persistent cart using localStorage
- Responsive design for all device sizes
- Clean and modern UI

## Files

- `index.html` - Homepage with featured products
- `products.html` - Product listing page with filtering
- `cart.html` - Shopping cart with summary
- `about.html` - About us page
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for cart and product management

## How to Use

1. Open `index.html` in a web browser
2. Browse products on the homepage or products page
3. Add items to your cart using the "Add to Cart" buttons
4. View your cart and update quantities or remove items
5. Proceed to checkout when ready

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for cart persistence

## Responsive Design

The website is fully responsive and works on mobile, tablet, and desktop devices.
