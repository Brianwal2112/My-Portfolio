# Dark Mode Portfolio

A simple, responsive portfolio website with dark/light mode toggle functionality, credential saving, and dedicated subpages for each section.

## Features

- Dark mode as default theme
- Toggle between dark and light modes
- Responsive design for all screen sizes
- Dedicated subpages for About, Skills, Projects, and Contact sections
- Individual project detail pages with comprehensive information
- Main landing page with previews and quick links
- Smooth navigation between pages
- Local storage to remember theme preference
- Credential saving functionality in contact form

## How to Use

1. Open the `index.html` file in any modern web browser
2. Click the moon/sun icon in the top right corner to toggle between dark and light modes
3. Navigate through the portfolio using the menu links to visit dedicated subpages
4. The theme preference will be saved in your browser's local storage
5. Fill out the contact form with your name and email to save credentials
6. On future visits, your name and email will be pre-filled in the form

## Subpages

Each section now has its own dedicated page:
- **About Page**: Detailed information about your background and journey
- **Skills Page**: Comprehensive list of your skills with proficiency levels
- **Projects Page**: Gallery of your projects with detailed descriptions
- **Contact Page**: Enhanced contact form with additional information

### Individual Project Pages

Each project has its own detailed page:
- **E-commerce Website**: Full details about the online store project
- **Task Manager App**: Comprehensive information about the productivity application
- **Weather Dashboard**: Detailed overview of the weather application
- **Fitness Tracker**: Information about the mobile fitness application
- **Blog Platform**: Details about the customizable blogging platform
- **Restaurant Booking System**: Information about the reservation system

## Customization

To customize the portfolio:

1. Edit the content in each HTML file to reflect your information
2. Modify colors in `styles.css` by changing the CSS variables
3. Adjust layout and styling as needed in `styles.css`

## Credential Saving

The contact form now includes credential saving functionality:
- When you submit the form with your name and email, they are saved to localStorage
- On subsequent visits, your name and email will be pre-filled in the form
- Credentials are stored locally in your browser and are not sent to any server

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
