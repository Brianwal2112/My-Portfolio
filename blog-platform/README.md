# BlogSpace - Blogging Platform

A comprehensive blogging platform with post creation, categorization, and reader engagement features.

## Features

- Home page with featured posts
- Blog listing with filtering and sorting
- Category browsing
- Article creation and management
- Post reading with comments
- Newsletter subscription
- Responsive design for all device sizes

## Files

- `index.html` - Home page with featured posts
- `blog.html` - Blog listing with filtering
- `categories.html` - Category browsing
- `about.html` - About page
- `contact.html` - Contact form
- `post.html` - Single post view
- `create-post.html` - Post creation form
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for blog management

## How to Use

1. Open `index.html` in a web browser
2. Browse featured posts on the home page
3. Navigate to the blog page to see all posts
4. Filter posts by category or sort by date/popularity
5. Create new posts using the create post form
6. Read individual posts and engage with comments
7. Subscribe to the newsletter for updates

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for data persistence

## Responsive Design

The platform is fully responsive and works on mobile, tablet, and desktop devices.
