# TaskFlow - Task Management Application

A comprehensive task management application with dashboard, task lists, calendar view, and team management features.

## Features

- Task dashboard with statistics
- Add, edit, complete, and delete tasks
- Filter tasks by status and priority
- Calendar view with task deadlines
- Team member management
- Local storage for data persistence
- Responsive design for all device sizes

## Files

- `index.html` - Dashboard with task statistics and quick add
- `tasks.html` - Complete task list with filtering
- `calendar.html` - Calendar view of task deadlines
- `team.html` - Team member management
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for task management

## How to Use

1. Open `index.html` in a web browser
2. Add new tasks using the quick add form
3. View all tasks on the Tasks page with filtering options
4. See task deadlines on the Calendar page
5. Manage team members on the Team page
6. Tasks are saved in your browser's local storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for data persistence

## Responsive Design

The application is fully responsive and works on mobile, tablet, and desktop devices.
