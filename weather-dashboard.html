Weather Dashboard

Frontend Development JavaScript, Weather API, Geolocation API

Project Overview

Real-time weather information with location services, forecasts, and severe weather alerts. This dashboard provides users with comprehensive weather data in an intuitive and visually appealing interface.

Features

  • Current weather conditions with detailed metrics
  • 5-day forecast with hourly breakdowns
  • Automatic location detection
  • Manual location search
  • Severe weather alerts and warnings
  • Interactive weather maps
  • Unit conversion (Celsius/Fahrenheit)

Challenges & Solutions

Integrating multiple weather APIs to provide accurate and timely information required careful data normalization. I implemented caching strategies to minimize API calls while ensuring data freshness.

Technologies Used

JavaScript Weather API Geolocation API CSS3 Chart.js Webpack Babel
