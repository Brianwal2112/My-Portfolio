# FitnessPro - Fitness Tracker

A comprehensive fitness tracking application with workout plans, nutrition logging, and progress tracking.

## Features

- Dashboard with daily fitness metrics
- Workout planning and tracking
- Nutrition logging and macro tracking
- Progress visualization with charts
- Achievement and goal tracking
- Data persistence using localStorage
- Responsive design for all device sizes

## Files

- `index.html` - Dashboard with fitness metrics and quick actions
- `workouts.html` - Workout plans and session history
- `nutrition.html` - Nutrition tracking and meal logging
- `progress.html` - Progress charts and goal tracking
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for fitness tracking

## How to Use

1. Open `index.html` in a web browser
2. Track your daily fitness metrics (steps, calories, etc.)
3. Plan and log workouts on the Workouts page
4. Log meals and track nutrition on the Nutrition page
5. Monitor your progress on the Progress page
6. Data is saved in your browser's local storage

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for data persistence

## Responsive Design

The application is fully responsive and works on mobile, tablet, and desktop devices.
