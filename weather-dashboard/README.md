# WeatherHub - Weather Dashboard

A comprehensive weather dashboard with current conditions, forecasts, maps, and alerts.

## Features

- Current weather conditions with detailed metrics
- Hourly and 5-day forecasts
- Interactive weather maps
- Weather alerts and notifications
- Location search functionality
- Responsive design for all device sizes

## Files

- `index.html` - Main dashboard with current weather and hourly forecast
- `forecast.html` - 5-day weather forecast
- `maps.html` - Interactive weather maps
- `alerts.html` - Weather alerts and notifications
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for weather data

## How to Use

1. Open `index.html` in a web browser
2. Enter a city name to get current weather conditions
3. View hourly forecast for the next 10 hours
4. Navigate to other pages for extended forecast, maps, and alerts
5. Use the search functionality to check weather in different locations

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Simulated weather data (would connect to real API in production)

## Responsive Design

The dashboard is fully responsive and works on mobile, tablet, and desktop devices.
