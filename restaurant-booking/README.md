# TableEase - Restaurant Booking System

A comprehensive restaurant booking system with reservation management, menu browsing, and guest services.

## Features

- Home page with restaurant highlights
- Online reservation system with date/time selection
- Digital menu with category filtering
- Contact and information pages
- Responsive design for all device sizes

## Files

- `index.html` - Home page with restaurant highlights
- `reservations.html` - Reservation booking form
- `menu.html` - Digital menu with categories
- `about.html` - Restaurant information
- `contact.html` - Contact form
- `styles.css` - Styling for all pages
- `script.js` - JavaScript functionality for booking system

## How to Use

1. Open `index.html` in a web browser
2. Browse the menu to see available dishes
3. Make a reservation using the booking form
4. Contact us with questions or special requests
5. View restaurant information on the about page

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- localStorage for data persistence

## Responsive Design

The booking system is fully responsive and works on mobile, tablet, and desktop devices.
